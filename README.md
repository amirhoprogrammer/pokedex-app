## Card list
![hover & list](https://github.com/amirhoprogrammer/pokedex-app/blob/master/public/New%20folder/Screenshot%202026-01-08%20161405.png)
When you scroll down the page, you will see a list of cards, and when you hover over the cards, you will see the card's shadow change.
![card list](https://github.com/amirhoprogrammer/pokedex-app/blob/master/public/New%20folder/Screenshot%202026-01-08%20161334.png)
As I said, I created the UI cards with Lucide React and got some of the card information from that site's API. This information includes the image, title, code, and label.
